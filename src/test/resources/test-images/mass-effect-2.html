Mass Effect 2

In this dark second chapter, humans are now faced with an even greater peril in this science fiction adventure set in a vast universe.

Commander Shepard returns in this epic space opera RPG, assembling a diverse team for a suicide mission. Experience refined combat, deeper character development, and moral choices that shape the galaxy's fate.

Key Features

  • Refined third-person shooter combat with biotic powers
  • Deep character loyalty missions and relationship building
  • Morality system affecting story outcomes and character fates
  • Squad-based tactical combat
  • Galaxy exploration with multiple planets and civilizations
  • Resource gathering and ship upgrade mechanics
  • Dialogue wheel system for meaningful conversations
  • Multiple romance options with complex relationships
  • Consequences carrying over from Mass Effect decisions
  • Epic suicide mission finale with multiple possible outcomes
